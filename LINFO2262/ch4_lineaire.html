🧒 Le neurone artificiel le plus simple

Le perceptron est le plus simple des classifieurs. Il simule le fonctionnement d'un neurone : il reçoit des signaux (les features xᵢ), les pondère (les poids wᵢ), les additionne, et si la somme dépasse un seuil → il s'active (prédit +1), sinon il reste silencieux (prédit -1).

Ce qui est remarquable, c'est la règle d'apprentissage : le perceptron ne se met à jour que quand il commet une erreur. Et il le fait toujours dans la bonne direction — en poussant les poids vers la solution correcte.

🎯 Imagines une balance à deux plateaux Tu dois décider si un fruit est une pomme ou une orange selon son poids et sa couleur.

• Poids w = [0.5, 0.3], biais b = -50. Entrée : [180g, 70% rouge].
• Score = 0.5×180 + 0.3×70 − 50 = 90 + 21 − 50 = 61 > 0 → prédit Pomme (+1) ✅

Si c'est vraiment une pomme : rien à faire.
Si c'est une orange : w ← w + η×(−1)×x. Les poids diminuent, le modèle apprend !

🔁 On répète pour chaque exemple, plusieurs epochs, jusqu'à convergence.
📥Entrées (x)Features : taille, poids, couleur...
⚖️Poids (w)L'importance de chaque feature — ce qui est appris.
➕Somme w·x + bLe score : positif ou négatif ?
🎯Décisionsigne(w·x+b) = +1 ou −1
  • 1. Initialise les poids w à zéro (ou aléatoirement).
  • 2. Pour chaque exemple (xˢ, yˢ) : si y·(w·x) ≤ 0 (erreur) → w ← w + η·y·x
  • 3. Répète jusqu'à convergence (ou nombre max d'epochs).
  • ⚠️ Limitation : converge seulement si les données sont linéairement séparables !

Interactif LMS / Widrow-Hoff — Descente de Gradient

🧒 Descendre dans la vallée à tâtons

Imagine que tu es perdu dans les montagnes par une nuit sans lune. Pour rejoindre la vallée (le minimum d'erreur), tu n'as qu'une seule stratégie : à chaque pas, avance dans la direction qui descend le plus. C'est la descente de gradient !

LMS (Least Mean Squares, aussi appelé Widrow-Hoff) est une règle d'apprentissage qui minimise l'erreur quadratique en suivant exactement ce principe. Contrairement au perceptron, il se met à jour même quand il ne fait pas d'erreur — proportionnellement à l'amplitude de l'erreur.

🎯 Perceptron vs LMS — quelle différence ?🔔 Perceptron :
• Se met à jour SEULEMENT si erreur binaire (mauvais signe)
• Mise à jour toujours de même amplitude : η·y·x
• Converge seulement si données linéairement séparables

📉 LMS :
• Se met à jour TOUJOURS, proportionnellement à l'erreur numérique
• Mise à jour : η·(y−ŷ)·x — plus l'erreur est grande, plus le pas est grand
• Converge vers la solution des moindres carrés même si données non séparables !

🏆 LMS est équivalent à maximiser la vraisemblance sous bruit gaussien (voir Ch.7 !)
⛰️Paysage d'erreurJ(w) = erreur quadratique. On veut le minimum.
📉GradientLa pente — dit dans quelle direction l'erreur monte.
👣Pas de gradientw ← w - η×gradient. On descend la pente.
🎯MinimumLa solution optimale w* = (XᵀX)⁻¹Xᵀy
LMS et la solution analytique convergent vers le MÊME résultat. LMS est utile quand les données sont trop nombreuses pour inverser la matrice. La solution analytique est exacte mais coûteuse pour de grandes matrices.

Contrairement au Perceptron (qui ne se met à jour que sur les erreurs binaires), LMS se met à jour sur tous les exemples avec une erreur continue.

Critère MSE
J(w) = (1/2) Σₛ (y^s - w^T x^s)²
Gradient
∇J(w) = -Σₛ (y^s - w^T x^s) · x^s
Mise à jour (online SGD)
w ← w + η · (y^s - w^T x^s) · x^s
Solution analytique
w* = (X^T X)^{-1} X^T y (Moindres Carrés)
LMS converge vers la solution des moindres carrés — identique à maximiser la vraisemblance sous bruit gaussien (Ch.7).

Concept Perceptron vs LMS — Comparaison

📊 Quand utiliser quoi ?

Tu as maintenant vu deux algorithmes pour les classifieurs linéaires. Voici comment choisir entre les deux :

🎯 Résumé de la comparaison🔔 Perceptron → Simple, rapide, mais limité aux données linéairement séparables. Historiquement important (Rosenblatt, 1958). La brique de base des réseaux de neurones.

📉 LMS / Widrow-Hoff → Minimise l'erreur quadratique continue. Converge même sur des données non séparables. Équivalent à la régression linéaire. Utilisé dans le traitement du signal (ADSL, WiFi !) et en deep learning (descente de gradient stochastique = SGD).

🚀 Dans la vraie vie : ni l'un ni l'autre n'est utilisé seul — on préfère les SVMs (Ch.5) ou les réseaux de neurones. Mais comprendre ces deux algos simples est essentiel pour comprendre tous les algos modernes !
Tout le deep learning moderne (GPT, Vision Transformers...) est basé sur la descente de gradient (SGD) — exactement le principe de LMS. Vous venez d'apprendre le cœur de l'IA moderne !
