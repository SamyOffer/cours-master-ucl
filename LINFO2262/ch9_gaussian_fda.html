1. Motivation — Estimer P(x|y)
Pourquoi les modèles Gaussiens ?

Le problème central

Le classifieur MAP est optimal : ŷ = argmax P(y) · P(x|y). Mais il faut estimer la vraisemblance P(x|y). Comment faire en grande dimension ?

🔵 Modèles Discriminatifs

Estiment directement les frontières de décision globalement (SVMs, Arbres, Deep Nets…) ou localement (k-NN).

🟢 Modèles Génératifs

Modélisent la distribution des données de chaque classe. Naive Bayes suppose l'indépendance. Le modèle Gaussien suppose une distribution normale multivariée.

Hypothèse Gaussienne

Tous les exemples de la classe y suivent une loi normale multivariée N(μy, Σy). On estime μ̂y et Σ̂y par maximum de vraisemblance sur chaque sous-ensemble Dy.

MAP : ŷ = argmax P̂(y) · f̂_y(x) y ∈ {1,…,C} MLE : μ̂_y = (1/|D_y|) Σ x_y^l ← moyenne empirique Σ̂_y = (1/|D_y|) Σ (x_y^l - μ̂_y)(x_y^l - μ̂_y)ᵀ ← covariance

2. Densité Gaussienne & Matrice de Covariance
Explorateur interactif des contours de la loi N(μ, Σ)

Densité Gaussienne multivariée (d dimensions)

La densité en x est donnée par la distance de Mahalanobis de x au centre μ, pondérée par la covariance Σ :

f_y(x) = 1 / sqrt((2π)^d |Σ_y|) · exp[ -½ (x-μ_y)ᵀ Σ_y⁻¹ (x-μ_y) ] └────────────────────────────┘ Distance de Mahalanobis²

Comparaison des 3 types de Σ et leur lien avec Naive Bayes

L'hypothèse diagonale revient à supposer l'indépendance conditionnelle des coordonnées → c'est le Naive Bayes Gaussien ! Elle réduit les paramètres de O(d²) à O(d) par classe.

Identité : Σ = σ²I → 1 paramètre/classe, contours circulaires Diagonale : Σ = diag(σ₁², …, σd²) → d paramètres, contours en ellipse axiale Pleine : Σ quelconque → O(d²) paramètres, ellipses orientées

3. Frontières de Décision Gaussiennes
Hyperquadriques (cas général) vs Hyperplans (covariances égales)

Cas général : Σ₁ ≠ Σ₂ → QDA

La frontière est une hyperquadrique (ellipse, parabole, hyperbole…). Le terme quadratique −½ xᵀ(Σ₁⁻¹ − Σ₂⁻¹)x ne s'annule pas.

Covariances égales : Σ₁ = Σ₂ = Σ → LDA

Le terme quadratique s'annule. La frontière devient un hyperplan (frontière linéaire). C'est le Linear Discriminant Analysis (LDA).

Fonction discriminante (log-MAP) : g_y(x) = -½ (x-μ_y)ᵀ Σ_y⁻¹ (x-μ_y) - ½ ln|Σ_y| + ln P(y) Cas Σ₁=Σ₂=Σ : g(x) = g₁(x) - g₂(x) = wᵀ(x - x₀) avec w = Σ⁻¹(μ₁ - μ₂) et x₀ = (μ₁+μ₂)/2 − ln(P(y=1)/P(y=2)) / ((μ₁-μ₂)ᵀΣ⁻¹(μ₁-μ₂)) · (μ₁-μ₂)

4. Fisher Linear Discriminant (FDA)
Projection optimale pour discriminer les classes

Idée : projeter sur une droite

Pour tout vecteur w, la projection scalaire wᵀx réduit les données à 1D. FDA cherche la direction w qui maximise la séparation entre les moyennes projetées tout en minimisant la dispersion intra-classe.

Critère de Fisher : J(w) = (m̃₁ - m̃₂)² / (s̃₁² + s̃₂²) = wᵀ S_B w / wᵀ S_W w Matrices de dispersion : S_y = Σ_{x∈D_y} (x - m_y)(x - m_y)ᵀ ← scatter classe y S_W = S₁ + S₂ ← within-class scatter S_B = (m₁-m₂)(m₁-m₂)ᵀ ← between-class scatter Solution optimale : w = S_W⁻¹ (m₁ - m₂)

Équivalence FDA ↔ LDA ↔ Classifieur Gaussien

Fisher DA est identique au classifieur Gaussien avec covariances égales (LDA) lorsque Σ̂ = S_W et μ̂_y = m_y. Le classifieur Gaussien avec covariances quelconques s'appelle Quadratic Discriminant Analysis (QDA).

LDA, QDA, FDA, Naive Bayes Gaussien
Méthode Hypothèse sur Σ Frontière Paramètres Avantage
LDA Σ₁ = Σ₂ = Σ Hyperplan (linéaire) O(d²) partagé Robuste, peu de données
QDA Σ₁, Σ₂ quelconques Hyperquadrique O(Cd²) Plus flexible
Naive Bayes Gaussien Σ_y diagonale Quadrique (axiale) O(Cd) Peu de paramètres
Gaussien identité Σ = σ²I Hyperplan orthogonal 1 par classe Plus simple possible

Points clés à retenir : ▶ FDA = LDA = Gaussien avec Σ₁=Σ₂ (quand Σ̂ = S_W , μ̂_y = m_y) ▶ LDA → frontière linéaire : terme quadratique s'annule ▶ QDA → frontière quadratique : covariances différentes ▶ Diagonal Σ → Naive Bayes Gaussien (indépendance des features) ▶ Mahalanobis = distance pondérée par la covariance
